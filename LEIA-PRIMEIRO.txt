CORREÇÃO DO LINK PÚBLICO

O link agora usa o ID real do documento do paciente (paciente-...), e não o token.
Isso elimina a busca por token que estava retornando "acesso não localizado".

1. Apague o conteúdo antigo do seu repositório local, preservando somente a pasta .git.
2. Copie todos estes arquivos para o repositório.
3. No GitHub Desktop, confirme que src/components/ProfissionalDashboard.tsx foi alterado.
4. Commit: Usar ID real no link público
5. Push origin.
6. Na Vercel, confirme que o deploy foi criado a partir desse commit.
7. Abra um paciente já cadastrado e copie novamente o link. Ele deverá terminar em ?id=paciente-...

IMPORTANTE: se o link ainda terminar em ?id=token-..., a versão nova não foi publicada.
